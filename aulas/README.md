# 🎓 Aulas de Java

Esta pasta contém o material de apoio das aulas de Java desenvolvidas durante os estudos, organizadas por tópicos e módulos.
Aqui você encontrará códigos, exemplos, anotações de conteúdo teórico e prático, e arquivos que complementam o aprendizado da linguagem Java.

---

## 📂 Estrutura

Cada subpasta representa uma aula ou conjunto de aulas.  
Os arquivos podem incluir:
- Códigos de exemplo demonstrados em aula
- Anotações teóricas e práticas
- Exercícios resolvidos durante as explicações
- Arquivos de apoio para estudo


## 🎯 Objetivo

O objetivo desta pasta é servir como **referência rápida de aprendizado** para:
- Revisar conceitos vistos em aula
- Consultar exemplos práticos de código em Java
