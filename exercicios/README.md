# 💻 Exercícios de Java

Este repositório contém uma coleção de exercícios de Java, organizados por tópicos, desde lógica básica até Programação Orientada a Objetos (POO). Ele foi estruturado para facilitar estudos, prática e construção de portfólio profissional.

---

## 📂 Estrutura do Repositório

- `estrutura_sequencial/`  
  Exercícios de cálculos e operações matemáticas simples, usando apenas sequência de comandos.

- `estrutura_condicional/`  
  Exercícios que exploram estruturas condicionais (`if`, `else if`, `else`) para tomada de decisão.

- `laco_repeticao/`  
  Exercícios utilizando loops (`for`, `while`) para repetição de operações, validação de dados e cálculos iterativos.

- `poo/`  
  Exercícios de Programação Orientada a Objetos:  
  - `entities/` – Classes principais dos exercícios (Student, Produto, Funcionario, etc.)  
  - `application/` – Classes com método `main` para testar cada entidade  

---

## ✅ Boas Práticas

- Nomes de arquivos **claros e descritivos**, facilitando a leitura e entendimento rápido.  
- POO implementa **encapsulamento**, **getters/setters** e métodos claros.  
- Exercícios de lógica foram organizados em pastas por tema, permitindo aprendizado progressivo.

---

## 🎯 Objetivo

- Servir como **repositório de estudo** e prática de Java.  
- Oferecer uma estrutura **limpa e profissional** para portfólio.  
- Facilitar a referência rápida a exercícios de diferentes tópicos.
